## QA & Test Automation

End-to-end automation using **Cypress** with a full Page Object Model architecture:

- Page Object Model (POM) — selectors and interactions encapsulated per page class
- Fixture-driven test data — decoupled from test logic via JSON fixtures
- Mochawesome HTML reports
- CI/CD via GitHub Actions — runs on every push and pull request
- Retry strategy for flaky network-dependent tests
